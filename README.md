# Titanic Data Cleaning & Preprocessing

This project performs basic data cleaning and preprocessing on the Titanic dataset using Python and Pandas.

## Features
- Handle missing values
- Remove duplicate rows
- Convert data types
- Rename columns
- Save cleaned dataset

## Technologies Used
- Python
- Pandas

## Dataset
Titanic Dataset

## Output
Cleaned dataset ready for analysis.
