# Titanic Data Cleaning & Preprocessing

This repository contains a Python script that performs essential data cleaning and preprocessing on the classic Titanic dataset. The goal of this project is to transform the raw data into a clean, analysis-ready format by handling missing values, standardizing column names, and removing redundant information.

## 🚀 Features

The `titanic.py` script executes the following preprocessing steps:
- **Missing Value Imputation:** - Fills missing `Age` values with the mean age (rounded to one decimal).
  - Fills missing `Embarked` values with the most frequent port (mode).
  - Fills missing `Fare` values with the median fare.
- **Feature Selection:** Drops the `Cabin` column due to a high percentage of missing values.
- **Data Deduplication:** Identifies and removes duplicate rows.
- **Type Conversion:** Converts `Survived` to integer and `Pclass` to a categorical variable.
- **Standardized Naming:** Renames columns (e.g., `PassengerId` to `Passenger_ID`, `Sex` to `Gender`) for better readability and consistency.

## 📂 File Structure

- `titanic.py`: The main Python script containing the data cleaning logic.
- `tested.csv`: The original, raw Titanic dataset.
- `Cleaned_Titanic.csv`: The final, preprocessed dataset generated by the script.
